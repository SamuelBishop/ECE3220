COPY CONSTRUCTOR:
- Used when passing argument into function by value
- OR returning by value

Three different types of passings:
1) Value C/C++ -> Used in the copy constructor
2) Reference C++
3) Pointer C/C++

T foo(T t) // value (copy)
T& foo(T &t) // reference (alias name)
T* foo(T *t) // pointer (address)
